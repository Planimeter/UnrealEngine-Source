# UnrealEngine-Source
Notes for porting Source Engine content to Unreal Engine

## Hammer
Start `hammer.exe` from `$GAME/bin`.[1]

## Utilities
* [bspsrc](https://github.com/ata4/bspsrc)
* [VMF2OBJ](https://github.com/Dylancyclone/VMF2OBJ)
* [Blender](https://www.blender.org/)

## Blender
### Scale[2]
`0.025400`

### Forward[3]
`X Forward`

### Up[2]
`Z Up`
